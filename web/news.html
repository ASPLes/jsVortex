News

  • 2014/03/14: jsVortex 0.6.1 "Levels"

    New stable release that includes full support for BEEP over WebSocket. Now it is possible to run BEEP applications on the web without plugins.

    Several improvements and bugs were fixed.

    See all details at the release note.

  • 2012/08/22: jsVortex 0.5.4 "Seven Devils"

    New stable release that includes fixings while handling SEQ frames and a new extended API to handle XML content.

    See all details at the release note.

  • 2012/05/21: jsVortex 0.5.3 "I got mine"

    New stable release that includes some fixings to support latest Java Applet releases (version 7). Fixed byte length verification when broken (split in the wrong place) utf-8 content is received.

    See all details at the release note.

  • 2012/01/04: jsVortex 0.5.2 "Why not?"

    New stable release that includes some fixings and a new building process that allows delivering a release version (without any debug statement) and a debug version (that includes them).

    See all details at the release note.

  • 2011/07/12: jsVortex 0.5.1 "Circles"

    New stable release which adds support for Safari, Opera and Internet explorer 7 (along already supported browsers Google Chrome, Firefox, IE8, IE9).

    See all details at the release note.

  • 2011/06/24: jsVortex 0.4.2 "La Marea"

    Published new stable release including bug fixings.

    See all details at the release note.

  • 2011/03/21: jsVortex 0.4.1 "Uprising at trantor"

    First release with complete support for Firefox, Internet Explorer and Google Chrome using the new java socket connector.

    Many API updates and general improvements making jsVortex more usable for production environment.

    Updated packaging to release the product into a single .js compressed file for fast download.

    See all details at the release note.

  • 2009/05/19: First public release: jsVortex 0.2.1 "There's no spoon"

    This is the first public release of jsVortex mainly to show the progress done until now and to get feedback about the product. It comes with a complete regression test checking implemented features. This regression test panel was created with the impressive Dojo toolkit product. See all details at the release note.
